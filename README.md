<!-- Run Locally -->
### :running: Run Locally

Clonar el proyecto

```bash
  git clone https://github.com/Anng3l/PoliGym_Backend_DAWEB.git
```

Instalar las dependencias

```bash
  npm i
```

Correr el servidor

```bash
  npm run dev
```
